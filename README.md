# A Multivariate Linear Model for Agricultural Data of Iowa for Corn, Soybeans, and Wheat

## Abstract
This project investigates the joint relationship between Corn, Soybeans, and wheat yield in Iowa, particularly climatic variables such as precipitation, alongside anthropogenic factors like fertilizer use, and soil quality, to fill the gaps that already exist in the univariate models. We took data from the USDA NASS from 2020 to 2025, with a focus on soil quality, considering the covariance between corn, soybeans, and wheat yields. The model follows the form Y=XB+E, where Y is the multi-crop yield matrix, X as the matrix for environmental predictors, and E the error terms associated.
